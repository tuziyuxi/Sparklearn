# 概观

结构化流是一种基于Spark SQL引擎的可扩展且容错的流处理引擎。可以像表达静态数据的批处理计算一样表达流式计算。
Spark SQL引擎将负责逐步和连续地运行它，并在流数据继续到达时更新最终结果。
结构化流传输提供快速，可扩展，容错，端到端的精确一次流处理。
